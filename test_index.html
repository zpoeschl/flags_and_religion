About

Welcome to the website of group 4 where we summarize our use of machine learning to explore the potential predicitability of a country's religion based on the design of their flag.

For this project, we used the flags data set available through the UCI Machine Learning Repository. This data set is not large, consisting of only 194 instances (countries/flags) and 30 attributes (descriptors), making the size of our experiments simultaneously manageable and limited. This data set can be viewed with limited attributes on our Data page.

To explore this data set and apply machine learning, we used a jupyter notebook along with tool scikit-learn to apply a neural network, as well as pandas, and matplotlib.

Data used for this project was obtained from the flags data set available through the UCI Machine Learning Repository.
